# RD Segmentation Challenge

## Requirements
 - Ruby >= 2.2.4
 - Gem Bundler
 - postgresql
 - sqlite3

## Installation
- Clone the repository
- Execute ```bundle install```
- Execute ```rake db:migrate```

### Run Application
Execute ```rails server```

### Run Tests
Execute ```bundle exec rspec```
